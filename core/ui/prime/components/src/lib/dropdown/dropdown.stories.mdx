# Icon

This is a nice dropdown!

<Story story={Default} />

First import `DropdownModule` from `@lens/ui-prime-components`.

```html
<lens-dropdown id="foo" placeholder="test" options="options"></lens-dropdown>
```

```ts
const options = [
    { value: 1, label: "Item 1", disabled: false },
    { value: 2, label: "Item 2", disabled: false },
    { value: 3, label: "Item 3", disabled: false },
    { value: 4, label: "Item 4", disabled: false }
];
```

The properties used for `value`, `label` and `disabled` can be changed and are configured using `optionValue`, `optionLabel` and `optionDisabled` respectively.
